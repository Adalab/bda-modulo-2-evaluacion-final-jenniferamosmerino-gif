## Repositorio de evaluación bloque II

En este repositorio hay dos archivos principales llamados "Ejercicio_1. Creación de una base de datos" que es un archivo de vs code  y "Ejercicio_2. Base de datos Sakila." que es un archivo sql.
hay una capeta que contiene la información para crear la base de datos Sakila y dos documentos donde podemos ver el diagrama o esquema que se ha usado para hacer el ejercicio 2.

## Ejercicio 1. Creación de una bases de datos de películas extraídas de un API
Se puede ejecutar el archivo completo o ir ejecutándolo por bloques de código.
las primeras líneas descargan las libreías necesarias para ejecutar el código, las siguientes se dedican a obtener la información de la api y limpiarla para después crear la nueva base de datos y la tabla y por último introduci los datos en ella.
las últimas líneas de código son las consultas pedidas, las cuales se mostrarán en formato dataframe para facilitar su lectura.

## Ejercicio 2. Base de Datos Sakila

Para este ejercicio primero se deben ejecutar en My SQL los dos archivos de la carpeta, los cuales crearán la base de datos con las que vamos a trabajas, la tabla y los datos dentro de ella.
Una vez ejecutados los dos archivos y refrescado el SQL para que aparezca la tabla y la base de datos podremos abrir el archivo del ejercicio 2 donde están las consultas.
Se puede ejecutar cada consulta por separado o todas al mismo tiempo.

#### Se puede abrir uno de los documentos de diagrama para ver el esquema de las tablas de las bases de datos.
